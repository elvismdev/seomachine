# SEO Scoring Criteria

## SEO Quality Score (0-100)

### Category Weights

| Category | What It Measures |
|----------|------------------|
| Content | Word count vs target (2000-3000), keyword in first 100 words, topic depth |
| Structure | H1 count (exactly 1), H2 count (4-6+), keyword in headings ratio (33%+) |
| Meta | Title length (50-60 chars), description length (150-160 chars), keyword presence |
| Links | Internal links (3-5+), external links (2-3+), descriptive anchor text |
| Readability | Sentence length (<25 words avg), reading level (grade 8-10), paragraph structure |

### Grade Scale

| Score | Grade | Publishing Status |
|-------|-------|-------------------|
| 85-100 | A | Ready to publish |
| 70-84 | B | Minor improvements recommended |
| 55-69 | C | Significant improvements needed |
| <55 | D/F | Major revision required |

### Keyword Density Guidelines

| Density | Assessment |
|---------|------------|
| <0.5% | Under-optimized — keyword appears too rarely |
| 0.5-1.0% | Acceptable — could be stronger |
| 1.0-2.0% | Optimal — natural and well-distributed |
| 2.0-3.0% | Borderline — risk of over-optimization |
| >3.0% | Over-optimized — keyword stuffing risk |

### Search Intent Types

| Intent | Content Format | Key Signals |
|--------|---------------|-------------|
| Informational | How-to guides, explanations, tutorials | "how to", "what is", "guide" |
| Commercial | Comparison pages, reviews, "best of" lists | "best", "vs", "review", "compare" |
| Transactional | Product pages, pricing, signup forms | "buy", "price", "discount", "deal" |
| Navigational | Brand-specific pages, product features | Brand name, specific product name |

### Critical Placements

Keywords should appear in:
1. Page title (meta title)
2. H1 heading
3. First 100 words of content
4. At least 1/3 of H2 headings
5. Meta description
6. URL slug
7. Image alt text (at least one)
8. Conclusion/final paragraph
